✂️ Раскадровка субтитров стиль Sorokina

Вставь монолог или речь из ролика. Уберу слова-паразиты, разобью на короткие смысловые блоки (3–5 слов), в каждом выделю одно акцентное слово фирменным цветом и отмечу эмоциональные хуки.
🎬 Готовый ролик с этим стилем (распознаёт речь, убирает «эээ» и паузы, накладывает субтитры и лёгкий zoom на хуках) собирается в облаке: GitHub → вкладка Actions → «Смонтировать видео» → вставь ссылку на видео с Google Drive.

🎥 Обработать отдельное видео без привязки к плану

Любой ролик: говоришь в кадре, голос за кадром, без речи или смешанный. Распознаётся речь только этого файла, ставятся фирменные субтитры, кадр приводится к 9:16. Бесплатно — OpenAI не участвует.

Публикация в соцсетях

Отложенная публикация через защищённый прокси-сервер (Cloudflare Workers). API-ключи не хранятся в браузере.
MAX с августа 2025 недоступен для самозанятых и физлиц. Скопируй текст и опубликуй вручную.

Настройка прокси-сервера

Все API-ключи (Meta, VK, Telegram) хранятся в Cloudflare Workers Secrets — никогда не попадают в клиентский код.
Как развернуть воркер →
1. npm create cloudflare@latest sorokina-publish 2. Скопируй workers/publish.js из папки проекта 3. wrangler secret put META_TOKEN 4. wrangler secret put VK_TOKEN 5. wrangler secret put TG_BOT_TOKEN 6. wrangler secret put TG_CHANNEL_ID 7. wrangler deploy 8. Вставь URL воркера выше

Telegram — подключение

Как создать бота — 3 шага
1. Открой Telegram, найди @BotFather и напиши /newbot
2. Придумай имя и username для бота — получишь токен вида 7123456789:ABC...
3. Добавь бота администратором в свой Telegram-канал (Управление каналом → Администраторы)
Для публичных каналов подойдёт @username. Для приватных — нажми «Найти ID»

ВКонтакте — подключение

Токен группы — 3 шага
1. Открой свою группу ВКонтакте → Управление.
2. Работа с API → Ключи доступа → Создать ключ.
3. Отметь права «Управление сообществом», «Стена», «Фотографии», создай ключ и скопируй его (начинается с vk1.a...).
Определится автоматически по токену. Если нет — впиши вручную (например -123456789).
